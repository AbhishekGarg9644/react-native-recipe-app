<h1 align="center">🍽️ Recipe Finder App</h1>

## 📌 Overview
A full-stack mobile application that allows users to discover recipes, search by categories, and manage personalized favorites.

---

## 🚀 Features
- 🔍 Search recipes using external APIs
- 🍳 Browse recipes by categories
- 📖 View detailed recipe information
- ❤️ Save and manage favorite recipes
- 🔐 User authentication with email verification (Clerk)

---

## 🛠️ Tech Stack
- **Frontend:** React Native (Expo)
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **ORM:** Drizzle ORM
- **Auth:** Clerk
